About System Viewer

System Viewer

Version 1.0.0

System Viewer is a real-time system monitoring application that provides comprehensive insights into your system's performance metrics.

This web-based application allows you to monitor CPU usage, memory consumption, and disk utilization from any device with a web browser. The real-time updates and historical tracking help identify performance trends and potential issues.

Features
  • CPU Monitoring: Track CPU usage percentage, load averages, and per-core usage
  • Memory Tracking: Monitor RAM usage, available memory, and swap utilization
  • Disk Analysis: Track disk usage across all partitions and I/O statistics
  • Historical Data: View performance trends with historical graphs
  • Real-time Updates: All metrics update automatically via WebSockets
  • Responsive Design: Optimized for both desktop and mobile devices
Usage Instructions
Getting Started

Navigate through the system metrics using the sidebar menu. Each page provides detailed information about different aspects of your system:

  • Dashboard: Overview of all system metrics
  • CPU: Detailed CPU usage statistics and graphs
  • Memory: RAM and swap memory usage details
  • Disk: Storage usage for all partitions and I/O statistics
Auto-Refresh

All metrics automatically refresh every 5 seconds. You can toggle auto-refresh by clicking the "Auto-refresh" button in the navigation bar.

Reading the Graphs

Green indicators show healthy system state
Yellow/orange indicators show moderate load
Red indicators show high/critical system load

Technical Details
Backend Framework FastAPI (Python)
Frontend HTML, CSS, JavaScript, Bootstrap 5
Visualization Plotly.js
Real-time Updates WebSockets
System Metrics psutil (Python library)
Target Environment Linux/Unix systems (Debian-based)
Required Dependencies:
  • Python 3.11+
  • FastAPI
  • Uvicorn
  • psutil
  • Jinja2
  • WebSockets
